\chapter{Resources}

%#todo need to get way for parser to change this into a chapter outside of a part in html

Here are some useful resources for learning different aspects of special relativity, including spacetime diagrams which I choose not to use as a learning aid in this handbook.

There is one thing that you should take into account when using these resources compared to this handbook, and that is that in this handbook we have chosen for the primed frame to be moving in the z-direction throughout, as it was needed when in came to spherical coordinates, whereas it is more common to have the primed frame going in the x-direction.
So you just need to swap the x and z coordinates in transformation equations to get the same transformations.
